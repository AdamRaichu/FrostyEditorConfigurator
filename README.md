# Frosty Editor Configurator

A simple tool to configure Frosty Editor for your needs.

## Usage

1. Open the website at [adamraichu.github.io/FrostyEditorConfigurator/fe1063](https://adamraichu.github.io/FrostyEditorConfigurator/fe1063/).
2. Select the plugins and files you want to include.
3. Click the "Download" button.
4. Save the file to your desired location.
5. Extract the generated file. You just installed Frosty Editor!
